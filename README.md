**AnekaSort**
--------------

This programs will sort users data (can be manually inputted or computer generated) in numbers using 5 different methods.

List of supported methods :

 1. Bubble Sort
 2. Selection Sort
 3. Insertion Sort
 4. Quick Sort
 5. Counting Sort
